## HOW TO EXECUTE THE CODES

# Compile and execute the c code by typing the following commands :

- gcc image_greyscale_svd.c -o image_greyscale_svd -lm (for greyscale image)
- ./image_greyscale_svd
  
- gcc image_colored_svd.c -o image_colored_svd -lm (for both greyscale and color images)
- ./image_colored_svd

Also,download the stb_image.h and stb_image_write.h library files from the c_libs folder. 

# To run the python code type the following command :

-python3 image_svd.py (for both greyscale and color images)
